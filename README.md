# Household & Financial Data Analysis

## Overview
This repository contains a **data analysis project** using two datasets to explore and answer analytical questions. The project demonstrates skills in **data cleaning, processing, and visualization**, as well as generating actionable insights from structured data.

The analysis showcases the ability to handle multiple datasets, perform calculations, and summarize findings clearly.

## Objectives
The main objectives of this project are to:

- Perform **data cleaning and preparation**  
- Conduct analysis to answer specific questions  
- Create **tables and visualizations** to summarize insights  
- Document assumptions and reasoning for analytical decisions  
- Demonstrate proficiency with **Excel**, **Power BI**, or **R**

## Tools & Technologies
- **Excel** – primary tool for data analysis and calculations  
- **Power BI** – optional tool for advanced visualization  
- **R** – optional for data manipulation and visualization  
- **Jupyter Notebook** – if R code is included
